# Glosario

Vocabulario compartido del dominio y del proyecto. Mantén las definiciones cortas
y sin ambigüedad para que todo el equipo use los términos de la misma forma.

| Término        | Definición                                                                                     |
| -------------- | ---------------------------------------------------------------------------------------------- |
| Autenticación  | Proceso de verificar la identidad de un usuario comparando las credenciales ingresadas.        |
| Cobertura      | Porcentaje de código ejercitado por los tests, medido con JaCoCo.                              |
| Login          | Acción de autenticarse con email y contraseña para acceder al sistema.                         |
| Mock           | Objeto simulado que reemplaza una dependencia real en un test (aquí, con Mockito).             |
| Registro       | Alta de un nuevo usuario en el sistema, previa validación de email y contraseña.               |
| Surefire       | Plugin de Maven que ejecuta los tests y genera el reporte de resultados.                       |
| Usuario        | Entidad con email y contraseña que puede registrarse y autenticarse.                           |

> Convención: ordena los términos alfabéticamente y enlaza al documento donde se
> detalla cada concepto cuando aplique.
